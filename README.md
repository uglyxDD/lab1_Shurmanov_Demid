## Задача 1.1 Дробная часть числа  

**Формулировка:**  

1. Привести число `x` к целому типу `(int)`.


## Задача 4.1 Диапазон  

**Формулировка:**  
```java
public boolean isInRange(int a, int b, int num);
```  
Метод возвращает `true`, если `num` входит в диапазон `[a, b]` (в любом порядке).  

**Алгоритм решения:**
1. Определить минимум и максимум из `a` и `b`.  
2. Проверить условие `min <= num <= max`.  

**Код реализации:**
```java
public boolean isInRange(int a, int b, int num) {
    int left = Math.min(a, b);
    int right = Math.max(a, b);
    return num >= left && num <= right;
}
```

**Примеры работы:**
```
a = 5, b = 1, num = 3   → true
a = 2, b = 15, num = 33 → false
```

## Задача 6. Абсолютное значение  

**Формулировка:**  
```java
public int abs(int x);
```  
Метод возвращает модуль числа `x` (через `if` и `switch`).  

**Алгоритм решения:**
1. Определить знак числа через `if`.  
2. Использовать `switch`, чтобы вернуть правильное значение.  

**Код реализации:**
```java
public int abs(int x) {
    int sign;
    if (x < 0) sign = -1;
    else if (x > 0) sign = 1;
    else sign = 0;

    switch (sign) {
        case -1: return -x;
        case  1: return x;
        case  0: return 0;
        default: return x;
    }
}
```

**Примеры работы:**
```
x = -7 → 7
x =  5 → 5
x =  0 → 0
```

```java
        case 3: return false;
        default: return false;
    }
}
```

```java

```  
Метод возвращает строку с числами от 0 до x включительно.  

**Алгоритм решения:**
1. Использовать цикл `for (int i=0; i<=x; i++)`.  
2. Добавлять числа в строку.


```java

```  
Метод возвращает количество цифр в числе `x`.  

**Алгоритм решения:**
1. Если `x < 0` → взять модуль.  
2. Если `x == 0` → вернуть 1.  
3. Делить на 10, пока `x > 0`, и считать шаги.  

**Код реализации:**
```java
public int numLen(long x) {
    if (x < 0) x = -x;
    if (x == 0) return 1;

        x = x / 10;
        count++;
    }
    return count;
}
```

```java
        for (int j = 0; j < x; j++) {
            System.out.print(square[i][j]);
        }
```

## Задача 19. Число Фибоначчи  

**Формулировка:**  
```java
public int fibonacci(int n);
```  
Метод возвращает n-е число Фибоначчи.  

**Алгоритм решения:**
1. Первые два числа задать вручную (0 и 1).  
2. Использовать цикл для вычисления следующих.  

**Код реализации:**
```java
public int fibonacci(int n) {
    if (n == 0) return 0;
    if (n == 1) return 1;
    int a = 0, b = 1;
    for (int i = 2; i <= n; i++) {
        int temp = a + b;
        a = b;
        b = temp;
    }
    return b;
}
```

**Примеры работы:**
```
n = 0 → 0
n = 1 → 1
n = 6 → 8
```

## Задача 20. Простое число  

**Формулировка:**  
```java
public boolean isPrime(int n);
```  
Метод проверяет, является ли число простым.  

**Алгоритм решения:**
1. Простое число больше 1 и делится только на 1 и само себя.  
2. Проверить делители от 2 до √n.  

**Код реализации:**
```java
public boolean isPrime(int n) {
    if (n <= 1) return false;
    for (int i = 2; i * i <= n; i++) {
        if (n % i == 0) return false;
    }
    return true;
}
```

**Примеры работы:**
```
n = 7  → true
n = 10 → false
```
